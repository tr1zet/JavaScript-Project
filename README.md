## 📋 О проекте

**World Countries Explorer** — это современное веб-приложение, разработанное в рамках курсовой работы. Приложение позволяет пользователям исследовать информацию о странах мира: население, площадь, столицы, языки и многое другое.

## ✨ Возможности

### 🔐 Страница приветствия
- Ввод имени пользователя
- Сохранение в localStorage
- Анимация появления и эффекты при наведении
- Отправка по клавише Enter

### 🌐 Главная страница
- Приветствие с именем пользователя
- Карточки стран с флагами
- Информация: столица, население, площадь, регион, языки

### 🔍 Фильтрация и сортировка
- **Поиск** по названию (русский/английский)
- **Фильтр** по регионам: Европа, Азия, Африка, Америка, Океания
- **Фильтр** по языкам
- **Сортировка** по названию, населению, площади

### 📊 Статистика
- Количество отображаемых стран
- Общее население
- Средняя площадь
